# Central CCTV - Configuración

Este archivo contiene configuraciones adicionales para la aplicación.

## Variables de Entorno

```
NODE_ENV=production
APP_NAME=Central CCTV
APP_VERSION=1.0.0
```

## URLs Importantes

- **Sitio en Vivo**: https://jgonzalez-iss.github.io/central-cctv/

## Notas de Desarrollo

- La aplicación usa LocalStorage para persistencia de datos
- Las credenciales se almacenan localmente
- No requiere backend o base de datos
- Compatible con navegadores modernos (ES6+)
